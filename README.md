# PWMTestBoard
- Arduino Board that controls a PWM output with a joystick input.

## Debugging
- Change DEBUG_MODE to true to start debugging
- Change DEBUG_MODE to false to return to normal operations

## Wiring
| Wire  | Arduino Port |
| ------------- | ------------- |
| Power  | 5V  |
| Ground  | GND  |
| Input Signal | A0 |
| Output PWM Sigal | 5 |
