# PWMTestBoard
- Arduino Board that controls a PWM output with a joystick input.

## Debugging
- Uncomment all Serial lines and change delay from 15ms to 100ms. 

## Wiring
| Wire  | Arduino Port |
| ------------- | ------------- |
| Power  | 5V  |
| Ground  | GND  |
| Input Signal | A0 |
| Output PWM Sigal | 5 |
